# カメラモジュール仕様

## 概要
このモジュール (`camera.py`) は、カメラからのビデオストリームをキャプチャして表示する基本的な機能を提供します。OpenCVライブラリ (`opencv-python`) を使用して、デフォルトのカメラにアクセスし、そのフィードをウィンドウに表示します。

## 機能
- デフォルトカメラを初期化し、開きます。
- カメラからフレームを継続的にキャプチャします。
- キャプチャしたフレームを「Camera Feed」というタイトルのウィンドウに表示します。
- ユーザーが 'q' キーを押すことでアプリケーションを終了できます。
- カメラアクセスおよびフレーム読み取りに関する基本的なエラー条件を処理します。

## 依存関係
- `opencv-python`: カメラアクセスと画像処理に必要です。

## 使用方法 (Windows環境)
1. Python仮想環境がセットアップされ、アクティブ化されていることを確認してください。
2. `opencv-python` パッケージをインストールします: `pip install opencv-python`
3. スクリプトを実行します: `python camera.py`
4. カメラフィードを表示するウィンドウが表示されます。閉じるには 'q' を押します。
